# Modelo:
Life_code = """
data {
    int N;
    real Life[N];
    real IDH[N];
    real Thinness[N];
}
parameters {
    real b0;
    real b1;
    real b2;
    real<lower=0> sigma;
}
model {
    for (i in 1:N)
        Life[i] ~ normal(b0 + b1 * IDH[i] + b2* Thinness[i], sigma);
}"""

El siguiente modelo se compone de tres partes: data, parametros y el modelos. La primera parte entrega al modelo los datos de entrada tanto para las variables independientes como las dependientes, la segunda parte son los valores que deseamos estimar, en este caso nos interesa estimar los valores de intersecciÃ³n (b0) y las pendientes (b1,b2) de las varibles, en conjunto con el valor de sigma. Finalmente en tercer lugar se define el modelo bajo el cual se estimarÃ¡n los parametros.
